These are few examples to show how easy it is to build Aria Ready UI Widgets. This used Twitter Bootstrap. Like Bootstrap, there are plenty of Libraries like Angular JS, Yaml, Zurb Foundation etc.. There main focus is on cross devices, but not cross users. It would be lovely if they can give the Most out of this rich experience to People With Disabilities (PWD) who used Assistive Technologies(AT) like Screen readers, Magnifiers etc. Yes Not Just Cross Browsers and Cross Devices but Cross Users as well..
