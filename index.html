These are few examples to show how easy it is to build Aria Ready UI Widgets. These examples uses Twitter Bootstrap. Like Bootstrap.
There are plenty of UI Framework Libraries like this; UI BootStrap by Angular JS, Yaml, Zurb Foundation, etc.. You could track some of them here at JSDB , which is a collection of JavaScript Libraries and of course on github..

The problem with these libraries, is that they are not specifically designed to be accessible from the start. The result is a negative impact across millions of pages across the web, as inaccessible components are repeatedly shared and incorporated into new projects daily.
Their main focus is on cross devices (RWD) , but not cross users (Responsible Web Design).
It would be lovely if they can give the Most out of this rich experience to People With Disabilities (PWD) who used Assistive Technologies(AT) like Screen readers, Magnifiers etc. Yes Not Just Cross Browsers and Cross Devices but Cross Users as well..

These examples are a clone of Twitter Bootstrap 2.2.2 and Accessibillity fixes applied on top of them. The Idea is based on the work by The Paciello Group (TPG). When they felt that the JQuery UI was not Accessibile. They showcased these examples ((http://hanshillen.github.com/jqtest/#goto_slider) ) . As a result, most of the JQuery examples are accessibile now.
